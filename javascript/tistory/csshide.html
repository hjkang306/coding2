CSS로 요소를 안보이게 하는 방법

css에서 요소를 안보이게 하는 방법은 여러가지가 있습니다. 잘 알아두었다가, 상황에 맞는 속성을 사용하도록 합니다.

/* 요소를 안보이게 하는 방법 1. display: none; (애니메이션 X, 영역X) 2. opacity: 0; (애니메이션 O, 영역 O) 3. visibility: hidden; (애니메이션 X, 영역X) 4. transform: scale(0); (애니메이션 O, 영역X) 5. width: 0; height: 0; (애니메이션 X, 영역X) : blind */
