indexOf()를 이용하여 CSS 속성 검색하기

JAVASCRIPT 메서드 검색하기

  • indexOf() : indexOf()는 문자열에서 특정 문자의 위치를 찾습니다.
  • slice() : slice()는 문자열에서 원하는 값을 추출합니다.
  • substring() : substring()은 문자열에서 원하는 값을 추출합니다.
  • substr() : substr()은 문자열에서 원하느 값을 추출합니다.
  • split() : split()는 문자열에서 원하는 값을 추출합니다.
  • trim() : trim()은 문자열의 앞뒤 공백을 제거합니다.
