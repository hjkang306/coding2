HTML 태그 검색하기

  • a : a 태그는 다른 페이지 이동을 설정합니다.
  • abbr : abbr 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • acronym : acronym 태그는 줄임말이나 머리글자를 표현하는 데에 사용합니다.
  • address : address 태그는 가까운 HTML 요소의 사람, 단체, 조직 등에 대한 연락처 정보를 나타냅니다.
  • applet : applet 태그는 자바 애플릿을 보이게 하는 곳에 사용됩니다.
  • area : area 태그는 이미지의 핫스팟 영역을 정의하고, 하이퍼링크를 추가할 수 있습니다. map 태그 안에서만 사용할 수 있습니다.
  • article : article 태그는 문서, 페이지, 애플리케이션, 사이트 안에서 독립적인 구획을 만들 때 사용합니다.
  • aside : aside 태그는 문서의 주요 콘텐츠와 연관성을 가지고 있지만, 해당 콘텐츠와 분리시킨 영역을 만들고 싶을 때 사용합니다.
  • audio : audio 태그는 사용됩니다.
  • b : b 태그는 사용됩니다.
  • base : base 태그는 사용됩니다.
  • bdi : bdi 태그는 사용됩니다.
  • bdo : bdo 태그는 사용됩니다.
  • bgsound : bgsound 태그는 사용됩니다.
  • big : big 태그는 사용됩니다.
  • blink : blink 태그는 사용됩니다.
  • blockquoto : blockquoto 태그는 사용됩니다.
  • body : body 태그는 문서의 콘텐츠 영역을 정의합니다.
  • br : br 행을 바꾸고 싶을 때 사용합니다.(줄바꿈 기능)
  • button : button 태그는 클릭이 가능한 버튼을 만듭니다.
